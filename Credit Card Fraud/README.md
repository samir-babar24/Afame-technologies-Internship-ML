# Credit Card Fraud Detection

Detect fraudulent credit card transactions using machine learning models.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Usage

1. Place your `train.csv` and `test.csv` in the project root.
2. Run:
```
python main.py
```
3. Trained model and scaler will be saved as:
- `fraud_model.joblib`
- `scaler.joblib`
